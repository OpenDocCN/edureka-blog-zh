# 选择正确的 NoSQL 数据库

> 原文：<https://www.edureka.co/blog/choosing-the-right-nosql-database/>

NoSQL 包括各种不同的数据库技术，是由于存储的数据量激增而开发的。关系数据库无法处理如此庞大的数据量，面临着灵活性的挑战。

这就是 NoSQL 数据库发挥作用的地方，它因其特性而广受欢迎。
