## **PIP 是什么？**

Pip 是所有 python 包和[模块](https://www.edureka.co/blog/python-modules/)的包管理器。我们使用 pip 在 python 中安装不同的包。对于 3.4 以上的版本，已经安装了 pip。它提供了一个简单的方法来设置工作环境。Pip 是“pip 安装包”的递归首字母缩写。

让我们看看如何在我们的系统上安装它。
