## **PMP 考试题目&答案**

1.  ### **Define the project scope. T3】**

    1.  它是用来完成项目工作的实验设计。
    2.  完成项目工作需要成本和进度的结合。
    3.  它是对完成项目所需工作的描述。【答案】
    4.  它是对完成项目所需的工作和资源的描述。

    1.  一项非常庞大而复杂的工程。
