## **Java 中的矩阵乘法**

通过使用二元运算从两个矩阵的元素中获得单个矩阵称为矩阵乘法。更简单地说，如果将 a*b 和 b*c 阶的两个矩阵 R 和 S 相乘，所获得的矩阵是 a*c 阶的。在 java 中，可以通过使用各种方法来高效地完成矩阵的乘法。下面讨论最有效的方法。

继续这篇文章

**使用 For 循环**

在这种方法中，我们使用 for 循环。

```
public class Main{
public static void main(String args[]){
//creating two matrices
int m1[][]={{1,2,3},{4,5,6},{2,3,4}};
int m2[][]={{1,2,3},{4,5,6},{2,3,4}};
int m[][]=new int[3][3]; //3 rows and 3 columns
//multiplying
for(int i=0;i<3;i++){
for(int j=0;j<3;j++){
m[i][j]=0;
for(int k=0;k<3;k++)
{
m[i][j]+=m1[i][k]*m2[k][j];
}
//end of k loop
System.out.print(m[i][j]+" "); //printing matrix
}
//end of j loop
System.out.println();
}
}}

```

**输出**

15 21 27

36 51 66

22 31 40

## **通过键盘**指定输入

```
import java.util.Scanner;
public class Main
{
public static void main(String args[])
{
int n;
Scanner input = new Scanner(System.in);
System.out.println("Enter base of matrices");
n = input.nextInt();
int[][] m1 = new int[n][n];
int[][] m2 = new int[n][n];
int[][] mat = new int[n][n];
System.out.println("Enter the elements of 1st matrix row wise : n");
for (int i = 0; i < n; i++)
{
for(int j = 0; j < n; j++)
{
m1[i][j] = input.nextInt();
}
}
System.out.println("Enter the elements of 2nd matrix row wise : n");
for (int i = 0; i < n; i++)
{
for(int j = 0; j < n; j++)
{
m2[i][j] = input.nextInt();
}
}
System.out.println("Multiplying the matrices : ");
for(int i = 0; i < n; i++)
{
for(int j = 0; j < n; j++)
{
for(int k = 0; k < n; k++)
{
mat[i][j] = mat[i][j] + m1[i][k] * m2[k][j];
}
}
}
System.out.println("Product :");
for (int i = 0; i < n; i++)
{
for(int j = 0; j < n; j++)
{
System.out.print(mat[i][j] + " ");
}
System.out.println();
}
input.close();
}
}

```
