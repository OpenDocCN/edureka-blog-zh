## **Java 是什么？****![](../Images/cce038d310280d95417f154ddd48ac78.png)T7】**

Java 是一种面向对象的编程语言。它现在是世界上广泛使用的计算机编程语言，用于构建 web 和桌面应用程序。它是满足许多组织的企业编程需求的首选语言，也已成为实现基于 Internet 的应用程序和设备软件的首选语言。
