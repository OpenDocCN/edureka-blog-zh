## **SQL 中的 IF 条件**

IF()函数用两个参数传递，一个为 true，另一个为 false。如果条件为真，函数返回一个值，如果条件为假，则返回另一个值。

### **SQL 中 IF 语句的语法:**

IF(条件，值 _ IF _ 真，值 _ IF _ 假)

### **参数值**

| *条件* | 必选。要测试的值 |
| *值 _ 如果 _ 真* | 可选。如果 *条件* 为真则返回值 |
| *value_if_false* | 可选。如果 *条件* 为假则返回值 |
